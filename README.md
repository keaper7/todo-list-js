# 🎯 Интерактивный Трекер Задач (To-Do List)

Простой, чистый и отзывчивый менеджер задач, написанный на чистом JavaScript (Vanilla JS) без использования сторонних библиотек и фреймворков.

##  Реализованный функционал:
* **Умное добавление:** Задачи добавляются как по клику на кнопку, так и по нажатию клавиши `Enter`.
* **Валидация:** Реализована проверка на пустую строку (пустые задачи не создаются).
* **Интерактивность:** Выполнение задачи переключается в один клик с помощью динамического добавления CSS-классов (`classList.toggle`).
* **Контроль событий:** Удаление задач реализовано через безопасное управление событиями (`e.stopPropagation`), что предотвращает конфликты кликов в интерфейсе.
* **Оптимизация кода:** Повторяющаяся логика вынесена в отдельные функции согласно принципу DRY (Don't Repeat Yourself).

##  Использовал:
* HTML5
* CSS3 
* Vanilla JavaScript (DOM Manipulation, Event Handling)
